# Parquet Explorer

Online at: https://xiangpenghao.github.io/parquet-explorer

It compiles [parquet-rs](https://github.com/apache/arrow-rs) to WebAssembly and uses it to explore Parquet files.

![screenshot](doc/parquet-explorer.gif)
